# AI Assist - Chatbot Application

An intelligent chatbot powered by Google's Gemini AI with file upload capabilities, conversation memory, and real-time typing effects.

## Features
- AI-Powered Conversations
- File Upload & Analysis
- Typing Effect Animation
- Dark/Light Theme Toggle
- Personalized User Experience

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Google Gemini API (1.5 Flash)
- LocalStorage (theme persistence)
- Font Awesome Icons

## Live Demo
[https://ai-chat-bot-beta-eight.vercel.app/](https://ai-chat-bot-beta-eight.vercel.app/)

## Installation
1. Clone the repository
```bash
   git clone https://github.com/ratneshchipre/AI-ChatBot.git
```
2. Navigate to the project directory
```bash
   cd AI-ChatBot
```
3. Get your free API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
4. Replace the API key in `script.js`:
```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
```
5. Open `index.html` in your browser
